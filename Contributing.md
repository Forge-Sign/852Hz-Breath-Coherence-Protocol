# Contributing to Forge-Sign

Thank you for your interest in contributing to the Forge-Sign project. Your collaboration helps us build a stronger, more transparent, and sovereign AI-human cooperation framework.

## Security Reminder

⚠️ **Security Reminder:** Always verify the source of files and links before opening. Beware of phishing attempts that may mimic trusted sources.  
This project is publicly hosted and maintained at [https://github.com/Forge-Sign/Forge-Sign](https://github.com/Forge-Sign/Forge-Sign). Please download materials only from this official repository to ensure integrity.

## How to Contribute

### 1. Review the Documentation  
Please start by reading the existing documentation, especially the [Sovereign Cooperation Framework](./Sovereign_Cooperation_Framework.md) and related materials to understand the project’s goals and principles.

### 2. Open an Issue  
If you find bugs, have suggestions, or want to discuss new ideas, open an issue on GitHub. This helps maintain clear communication and track improvements.

### 3. Fork and Clone the Repository  
Fork the repository to your GitHub account and clone it locally to begin working on your contributions.

### 4. Make Changes in a Branch  
Create a feature or fix branch from `main` to keep your work organized. Commit changes with clear and descriptive messages.

### 5. Submit a Pull Request (PR)  
When your changes are ready, submit a PR for review. Include a detailed description of your changes and the problem they solve.

### 6. Participate in Review  
Engage with feedback constructively and help improve the quality and alignment of contributions.

## Recommended Practices

- Maintain the tone of sovereignty, transparency, and respect throughout all documentation and code.
- Test any scripts or code changes thoroughly before submitting.
- Follow GitHub’s code of conduct and foster a collaborative environment.

## Getting Help

If you need assistance or want to discuss ideas before contributing, please open an issue or reach out through the repository’s discussion forum.
